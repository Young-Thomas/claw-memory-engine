# Memory Forgetting Skill

当用户想要管理记忆的生命周期、查看遗忘状态或触发复习提醒时使用此技能。

## 触发条件

- 用户说"哪些记忆快忘了"、"检查过期记忆"、"复习提醒"
- 用户说"forgetting"、"review"、"expiring"
- 用户想了解记忆的健康状态

## 使用方式

### 手动检查即将过期的记忆

```bash
claw scheduler-check
```

当用户说"检查一下哪些记忆快过期了"时，执行此命令。

可以指定飞书群聊 ID 推送通知：

```bash
claw scheduler-check --chat-id oc_xxx
```

### 启动遗忘调度器

```bash
claw scheduler-start
```

当用户说"启动定时检查"时执行。调度器会在每天指定时间（默认9点）自动检查即将过期的记忆并通过飞书推送提醒。

## 遗忘曲线原理

系统基于 **Ebbinghaus 遗忘曲线** 管理记忆生命周期：

- **保留率公式**：R = exp(-t / S)
  - R：保留率（0-1）
  - t：经过时间（天）
  - S：稳定性系数（由使用频率决定）

- **记忆状态**：
  - 🟢 健康：保留率 > 50%
  - 🟡 需要复习：保留率 30%-50%
  - 🔴 即将过期：保留率 < 30%
  - ⚫ 已过期：保留率 < 10%

- **复习间隔**：1天 → 2天 → 4天 → 7天 → 15天 → 30天 → 60天 → 90天

- **频率加成**：使用越频繁，稳定性系数越高，遗忘越慢

## 飞书集成

遗忘调度器可以自动通过飞书推送提醒卡片：
- 即将过期的记忆会发送红色警告卡片
- 需要复习的记忆会发送橙色提醒卡片
- 用户可以点击"我已记住"按钮重置遗忘曲线
